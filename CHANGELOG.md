# 更新日志

本项目采用语义化版本，遵循 Keep a Changelog 的结构。

## [Unreleased]

### 新增
- Excalidraw 白板主题专项样式：补充 Island、对话框/菜单、侧栏、工具按钮、库切换器、输入框等完整配色与状态映射，统一为 Tokyo Night 语义色板与浮层阴影体系。
- Excalidraw 右键菜单可读性增强：hover 行文本增加高优先级亮色覆盖（含 `-webkit-text-fill-color`），修复暗色背景下文字发黑、对比不足的问题。
- 编辑区白板块新增柔和绿色轮廓：在 `.orca-repr-whiteboard-content` 容器上添加低存在感边框与微光，并在 hover 时轻微提亮作为交互提示。

### 改进
- 行内代码按键质感重做：引入亮/暗双主题 keycap 语义变量，统一顶面、台阶、投影与文本细节；亮色模式降低视觉侵略性，暗色模式弱化底部高亮边缘。
- 行内代码交互反馈优化：hover/active 调整为“顶面下压、阴影锚定”模型，修复此前位移/收缩观感偏差并提升按键触发真实感。
- 构建流程补强：新增 `postbuild` 自动复制 `public/tokyo-night.css` 到 `dist/tokyo-night.css`，确保构建产物样式始终同步。
- 侧边栏“新页面”按钮交互重做：移除 `+` 旋转语义，改为 `+` 与文字在 hover 时轻微放大与辉光反馈，同时保持按钮本体 hover 基调稳定。
- 优化“新页面”按钮 hover 稳定性：去除易触发抖动的滤镜组合，收窄文本动画作用域并补充渲染稳定参数，减少 hover 时抖动感。
- 通用实心按钮回退样式增强：为未命名 `solid` 按钮补充统一的常态/hover/disabled 视觉规范，减少弹层与编辑器内操作按钮风格漂移。
- 菜单与弹层容器可见性修正：针对多处菜单/弹层统一 `overflow` 与边框层处理，降低双层描边与遮挡导致的视觉噪声。
- 亮色模式“新页面”按钮视觉对齐修复：移除顶部高光偏移与 hover 背景亮度抖动，统一亮/暗模式下按钮本体的稳定表现。
- 代码块文本清晰度修复：消除亮色模式代码块文字白边/镶嵌感，统一代码阅读时的渲染观感。
- 日历日期 hover 交互重构：替换为轻量 Dock 风格缩放反馈（仅影响当前与相邻日期），并区分亮/暗模式下的对比度。
- 侧边栏与菜单项 hover 过渡平滑化：补充背景与文字颜色过渡，减少鼠标在上下项切换时的视觉跳变。
- 主编辑区加粗文本 hover 降强：减小缩放与发光强度，避免悬停时溢出。
- 主编辑区外框融合优化：右边界与下边界改为与窗口边界视觉融合，保留左侧悬浮层次与交互边界不变。
- 主编辑区容器定位修正：将主编辑区物理上边界锚定到工具栏下边界，避免仅通过内容内边距产生“伪对齐”。
- 主编辑区阴影分离调优：亮色模式增强浮层感，暗色模式改为更干净的单层阴影，降低浑浊感。

## [0.1.6] - 2026-03-12

### 新增
- 块展开/收起圆点视觉区分度增强：普通文本块展开为半透明灰色，收起为主题蓝色带发光效果
- 收起块圆点液体变形动画：切换折叠状态时触发挤压拉伸的液体变形效果
- 分割线联动悬停：悬停任意一侧水平线时，整个分割线同步点亮
- 分割线悬停区域扩展：伪元素覆盖 200% 宽度，更易触发悬停效果
- 行内代码样式：复刻 phycat-abyss 的键盘按键效果
- 按键采用立体边框设计（上亮下暗），配合双层阴影营造厚度感
- 悬停时按键下沉，阴影缩短，边框变为主题蓝色
- Callout 提示框样式：复刻 phycat-abyss 的 .md-alert 效果到 Orca 引述块
- 支持七种类型：Quote、Info、Warning、Error、Tip、Important、Note
- 采用从左到右的渐变背景（左侧透明 → 右侧类型色），避免边框超出范围问题
- 右下角 Emoji 图标，悬停时旋转放大并去除灰度
- 悬停时增强阴影效果，提供视觉反馈
