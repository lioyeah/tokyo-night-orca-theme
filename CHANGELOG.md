# 更新日志

本项目采用语义化版本，遵循 Keep a Changelog 的结构。

## [Unreleased]

### 新增
- 块展开/收起圆点视觉区分度增强：普通文本块展开为半透明灰色，收起为主题蓝色带发光效果
- 收起块圆点液体变形动画：切换折叠状态时触发挤压拉伸的液体变形效果
- 分割线联动悬停：悬停任意一侧水平线时，整个分割线同步点亮
- 分割线悬停区域扩展：伪元素覆盖 200% 宽度，更易触发悬停效果
- 行内代码样式：复刻 phycat-abyss 的键盘按键效果
- 按键采用立体边框设计（上亮下暗），配合双层阴影营造厚度感
- 悬停时按键下沉，阴影缩短，边框变为主题蓝色
- Callout 提示框样式：复刻 phycat-abyss 的 .md-alert 效果到 Orca 引述块
- 支持七种类型：Quote、Info、Warning、Error、Tip、Important、Note
- 采用从左到右的渐变背景（左侧透明 → 右侧类型色），避免边框超出范围问题
- 右下角 Emoji 图标，悬停时旋转放大并去除灰度
- 悬停时增强阴影效果，提供视觉反馈

### 改进
- 移除分割线菱形装饰，改为中间实心两端渐隐的连续视觉效果
- 移除块展开/收起圆点的 hover 效果，避免与宿主效果冲突
- Callout 块内隐藏块操作圆点，避免与左侧装饰线干涉
- 优化顶部工具栏与编辑区布局，编辑区上边界与工具栏下边界对齐（参考官方主题做法）
- 统一工具栏背景色与窗体控制按钮，消除视觉不一致
- 移除工具栏 mask-image 渐变效果，修复插件按钮背景显示异常问题
- 调整编辑器顶部内边距，改善首行内容的可读性

## [0.1.5] - 2026-01-08
- 重构：针对 "Vibe Coding" 全面优化 CSS 结构，引入 `--tn-level-1` 至 `6` 语义化层级变量。
- 新增：H1-H6 标题的分级颜色与渐变下划线效果（左透明 -> 右清晰）。
- 新增：无序与有序列表（UL/OL）的分级着色（红/橙/黄/绿/蓝/紫），完美覆盖 Orca 复杂 DOM 结构。
- 修复：强制对齐所有列表标记（包括 `::marker`、`::before` 伪元素及折叠箭头）的颜色。
- 优化：注释掉块操作手柄的默认颜色，使其能自然继承层级色。

## [0.1.4] - 2026-01-07
- 重构：将 Floating Shell 样式整合到主主题文件中，移除单独的 CSS 文件。
- 优化：侧边栏采用磨砂玻璃效果，透明背景与窗体自然融合。
- 优化：主编辑区悬浮显示，带圆角和阴影，视觉层次更清晰。
- 移除：删除 "Enable Floating Shell" 设置选项，悬浮效果默认启用。
- 简化：精简 main.ts 代码，移除条件注入逻辑。

## [0.1.3] - 2025-12-20
- 重构：全面移除硬编码 HEX 颜色值，统一使用 CSS 变量管理配色，提升维护性。
- 修复：修正设置模态框、侧边栏及搜索框的背景与文字颜色异常。
- 优化：深度定制日历组件样式，确保年份、月份及日期颜色符合 Tokyo Night 色板，去除不协调的白色元素。
- 优化：调整侧边栏标签栏及输入框的高亮与选中效果，视觉更柔和协调。
- 杂项：清理冗余代码、未使用的脚本及资源文件（如 icon.svg），简化构建流程。

## [0.1.2] - 2025-12-14
- 修复：统一主题 CSS 路径为 `dist/theme.css`，避免 `dist/dist` 造成的 404。
- 优化：插件目录结构对齐参考主题，运行时代码集中于 `dist/`。
- 优化：增强列表标记与编号的可读性，覆盖原生 `::marker`、容器伪元素 `::before`、Tabler Icons。
- 新增：分层配色（无序：青/春绿/洋红；有序：蓝/黄/红）以提升层级辨识度。
- 清理：移除内部列表扫描命令及相关注册，保持主题纯净。
- 构建：新增 `scripts/*` 脚本与 `dist/package.json` 生成；在 `.gitignore` 中忽略 `.deploy/`。
